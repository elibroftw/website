Cloud Copy

About

Cloud Copy [Abandoned] is a novel solution to clipboard management. It allows you to send copy text/content on one device, and paste it directly into another, seamlessly.

Cloud Copy differentiates itself from other clipboard applications by not depending on Chrome, or any browser you use. Versatile accross all platforms, connect all your devices with a single account.

Made by Elijah Lopez, Abdullah Hadi, Gijsson Yeo, and Kritik Kaushal. Source code.
